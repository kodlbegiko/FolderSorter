# Roadmap

FolderSorter 的 Roadmap 依照 [PRODUCT_STRATEGY.zh-TW.md](PRODUCT_STRATEGY.zh-TW.md)
制定：目標不是變成最強自動化工具，而是成為一般 Mac 使用者最敢按下 Apply 的
檔案整理工具。

## 最高優先級下一步

這些項目會優先做，因為 FolderSorter 應該先贏在信任感、可理解、可復原，
再往更深的自動化能力前進。

### 1. 信任優先的預覽與 Apply 流程

預覽畫面應該成為核心產品介面。使用者在任何檔案被處理前，就要看懂整理計畫。

交付項目：

- 依圖片、文件、壓縮檔、安裝檔、影片、未分類顯示總覽數量。
- Before / After 視圖，顯示原始位置與預計目標位置。
- 每個檔案顯示命中原因，例如副檔名、檔名關鍵字、檔案大類。
- 針對 Move 模式、大量檔案、同名衝突、未知類型、權限問題顯示風險提示。
- Apply 前最後確認，依 copy、move、100 個以上檔案、衝突、未分類檔案調整提示。

### 2. Undo 與整理報告

Undo 應該可靠到可以成為主賣點，而不只是安全按鈕。

交付項目：

- 完成報告顯示成功、跳過、失敗、新增資料夾、是否可復原的數量。
- 可檢視最近一次 transaction 紀錄。
- Undo 前顯示復原預覽。
- 復原失敗時，用人話說明原因。
- 強化 Apply 中斷處理，避免 transaction 狀態不明。

### 3. 發布與安裝信任

FolderSorter 已經有可下載的 `.zip` 與 `.dmg`。下一個 release 信任重點是降低
macOS 安全提示造成的阻力。

交付項目：

- 維持可重現的 `.app`、`.zip`、`.dmg` release script。
- 文件化未簽章 build 第一次開啟時的 macOS 安全提示。
- 加入 Apple Developer signing。
- 加入 notarization 與 stapling。
- 在乾淨 Mac 使用者環境重新測試安裝流程。

### 4. 強化視覺化 GUI 規則

規則要更強，但不能要求使用者寫 YAML、JSON 或 script。

預計加入的規則條件：

- 檔案大小
- 建立日期
- 修改日期
- 來源資料夾
- 是否為截圖
- 是否為重複檔
- 圖片、影片、文件、壓縮檔、安裝檔、音訊、程式碼等大類型

### 5. 第一次使用體驗

第一次使用應該在 1 分鐘內讓使用者從混亂資料夾走到預覽。

預計體驗：

- 起始入口提供 Downloads、Desktop、自訂資料夾。
- 提供安全等級：最安全 copy-only、標準 move-with-undo、進階自訂。
- 第一次整理流程不要求使用者理解規則語法或技術設定。

## 成熟度等級

### Level 1：可用工具

- 移動或複製檔案前先產生安全預覽
- 一鍵復原 ledger
- GUI 與 CLI
- JSON 規則匯入 / 匯出
- 同名衝突策略：自動改名、略過、取代
- 一般 Mac 混亂檔案的預設規則
- 本機優先處理

### Level 2：可信任工具

- 清楚 Before / After 預覽
- Apply 報告
- Undo 報告與 Undo 預覽
- 人話錯誤訊息
- 穩定 transaction 歷史
- 可安裝 Mac app
- 簽章與 notarization
- GUI 規則編輯
- 新手模式

### Level 3：好用產品

- 學生、辦公工作者、創作者、開發者規則模板
- 日期與大小條件
- 截圖偵測
- 大檔案整理
- 多資料夾整理
- 每個檔案可顯示命中原因
- 設定保存
- 完整英文與繁體中文介面
- 接近原生 Mac app 的視覺體驗

### Level 4：可成長平台

以下是未來方向，不是現在的重點：

- 根據資料夾內容建議智慧規則
- 安全預覽式重複檔案偵測
- 本機 PDF 文字分類
- EXIF 照片分類
- 可分享規則包
- 進階本機處理 plugin API
- 可控的排程整理，且仍維持 preview-first 確認
