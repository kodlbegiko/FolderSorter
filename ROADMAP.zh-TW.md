# Roadmap

## V1

- 移動或複製檔案前先產生安全預覽
- 一鍵復原 ledger
- GUI 與 CLI
- JSON 規則匯入/匯出
- 同名衝突策略：自動改名、略過、取代
- 一般 Mac 混亂檔案的預設規則

## V1.1

- 簽章 release builds
- 使用 hash 做更好的重複檔偵測
- 更多規則包：學生、創作者、辦公工作者、開發者
- 可選的排程掃描，但仍維持 preview-first 確認

## V2

- 依媒體長度、圖片尺寸、EXIF 日期分類
- 可選 OCR 與本機文字擷取
- 從近期手動整理紀錄建議規則
- 進階本機處理 plugin API
